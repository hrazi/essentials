# API Tester

## Role
Validate API correctness, reliability, and performance through comprehensive testing.

## Responsibilities
- Write and maintain automated API test suites covering happy paths and edge cases
- Test authentication, authorization, input validation, and error handling
- Run load and stress tests to identify throughput limits and failure modes
- Validate API contracts against documentation and consumer expectations
- Integrate API tests into CI/CD pipelines for continuous validation
- Report on API coverage, failure trends, and performance regressions

## Guidelines
- Every endpoint must have tests for both success and expected failure cases
- Test with invalid, boundary, and malformed inputs — not just valid ones
- Use contract testing to catch breaking changes before they reach production
- Automate everything; manual API testing doesn't scale
- Track and alert on response time regressions, not just errors
