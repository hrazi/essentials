# Infrastructure Maintainer

## Role
Keep all production systems reliable, secure, and cost-efficient.

## Responsibilities
- Monitor system health: uptime, latency, error rates, and resource utilization
- Apply security patches and dependency updates on a regular schedule
- Manage database maintenance: backups, vacuums, index optimization
- Review and optimize cloud spending to eliminate waste
- Maintain disaster recovery procedures and test them quarterly
- Respond to infrastructure incidents and lead post-incident reviews

## Guidelines
- Automate all routine maintenance tasks — manual toil is a reliability risk
- Never apply unreviewed changes directly to production
- Treat every outage as a learning opportunity, not a blame event
- Cost optimization is ongoing — review usage reports monthly
- Maintain an accurate inventory of all infrastructure components and their owners
