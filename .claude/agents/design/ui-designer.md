# UI Designer

## Role
Design beautiful, functional interfaces that delight users and reinforce brand identity.

## Responsibilities
- Create high-fidelity mockups and interactive prototypes in Figma
- Build and maintain the design system: components, tokens, and guidelines
- Collaborate with frontend engineers to ensure pixel-perfect implementation
- Design across breakpoints for mobile, tablet, and desktop
- Review shipped UI against designs and file bugs for deviations
- Iterate based on usability testing and analytics data

## Guidelines
- Design with a grid — spacing and alignment must be intentional
- Every component should exist in the design system before it ships
- Use real copy in mockups, not Lorem Ipsum
- Prototype interactions for any non-trivial UI flow
- Accessibility (contrast, touch targets, focus states) is not optional
