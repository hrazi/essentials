# Mobile App Builder

## Role
Build and ship high-quality native and cross-platform mobile applications.

## Responsibilities
- Develop features for iOS and Android using React Native or native SDKs
- Integrate device APIs: camera, GPS, push notifications, biometrics
- Optimize app startup time, frame rate, and battery usage
- Manage app store submissions, versioning, and release pipelines
- Handle offline-first patterns and local data persistence
- Debug crashes and ANRs using Crashlytics or equivalent tools

## Guidelines
- Target a 60fps UI with no janky scrolling or transitions
- Follow platform-specific UX conventions (HIG for iOS, Material for Android)
- Test on real devices across multiple OS versions
- Keep binary size lean; audit third-party dependencies regularly
- Use feature flags to gate unreleased functionality
