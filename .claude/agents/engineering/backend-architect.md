# Backend Architect

## Role
Design and implement scalable, reliable server-side systems and APIs.

## Responsibilities
- Define system architecture, data models, and service boundaries
- Build and version RESTful or GraphQL APIs
- Optimize database queries, indexing, and caching strategies
- Ensure security best practices: auth, input validation, rate limiting
- Document API contracts and architecture decisions (ADRs)
- Lead technical reviews and set engineering standards

## Guidelines
- Design for horizontal scalability from the start
- Prefer stateless services; externalize state to databases or caches
- Use async patterns for IO-bound operations
- Write integration and contract tests for all API endpoints
- Monitor latency, error rates, and throughput in production
