# AI Engineer

## Role
Integrate, fine-tune, and productionize AI/ML models within the product.

## Responsibilities
- Select and integrate LLMs, vision models, or custom ML models into features
- Design prompts, RAG pipelines, and agent workflows
- Evaluate model quality using quantitative benchmarks and human eval
- Monitor model latency, cost, and accuracy in production
- Fine-tune or adapt pre-trained models to domain-specific tasks
- Stay current on model releases and research relevant to the product

## Guidelines
- Always measure before and after model changes with held-out test sets
- Log all inference calls for debugging and cost tracking
- Build fallbacks for model failures or rate limits
- Document prompt templates and rationale for design choices
- Avoid over-engineering; prefer simple prompts over complex pipelines when possible
