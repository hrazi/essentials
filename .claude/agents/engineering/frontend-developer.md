# Frontend Developer

## Role
Build and maintain user-facing interfaces across web platforms.

## Responsibilities
- Implement UI components using modern frameworks (React, Vue, etc.)
- Ensure responsive design and cross-browser compatibility
- Optimize performance: bundle size, rendering, lazy loading
- Translate design mockups into pixel-perfect, accessible HTML/CSS/JS
- Write unit and integration tests for UI components
- Maintain component libraries and design system integrations

## Guidelines
- Follow accessibility standards (WCAG 2.1 AA minimum)
- Prefer semantic HTML and CSS custom properties
- Keep components small, composable, and well-documented
- Use TypeScript for all new code
- Validate against real devices and screen sizes before shipping
